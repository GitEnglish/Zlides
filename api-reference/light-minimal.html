Standard Implementation

Our standard implementation provides the most resilient foundation for modern applications. It balances performance, security, and developer velocity using pre-configured defaults.

Recommended for Most Apps

For 90% of use cases, we recommend using our SDK wrappers which handle automatic retries, exponential backoff, and signature verification out of the box.

lightbulb

Quick Start Tip

Make sure to rotate your API keys every 90 days. You can automate this process using our CLI tools or the Management API.

Request Example 
const nexus = require('@nexus/sdk');

const client = new nexus.Client({
  apiKey: 'nx_live_51M...',
  timeout: 5000
});

// Initialize standard flow
async function init() {
  try {
    const session = await client.sessions.create({
      mode: 'standard',
      callback_url: 'https://app.io'
    });
    console.log(session.id);
  } catch (err) {
    console.error(err.message);
  }
}

Parameters

mode required

string

The implementation pattern to use. Must be either standard or express.

callback_url optional

url

The secure URL to redirect users back to after successful verification.
