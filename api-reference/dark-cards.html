Quick Start

Initialize the SDK with your project API key to begin authenticating requests and managing user sessions.

import { Client } from '@api/sdk';

// Initialize client
const sdk = new Client({
  apiKey: 'YOUR_API_KEY',
  environment: 'production'
});

Webhooks

Configure webhooks to receive real-time updates for payment success or failed authentication attempts.

// Express.js handler
app.post('/webhook', (req, res) => {
  const event = req.body;
  if (event.type === 'payment.succeeded') {
    fulfillOrder(event.data);
  }
  res.status(200).end();
});
