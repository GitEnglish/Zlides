Quick Start Guide

Welcome to the DevPortal API. This guide will help you get integrated and make your first authenticated request in less than 5 minutes.

1

Install the SDK

First, install our official library using your preferred package manager.

terminal
npm install @devportal/core-sdk
2

Production Configuration

Initialize the client with your secret API key. Make sure to keep this key safe and never expose it in client-side code.

javascript 
// Initialize client
const { DevPortal } = require('@devportal/core-sdk');

const client = new DevPortal({
  apiKey: 'sk_live_...',
  environment: 'production'
});

// Verify connection
await client.authenticate();
info

Need a production key?

You can find your production keys in the Dashboard settings.
